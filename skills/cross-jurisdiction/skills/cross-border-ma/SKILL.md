# /cross-border-ma

## Instructions

1. 中国法部分关注外资、审批备案、数据/劳动/知识产权及交割条件。
2. 美国法部分关注公司治理、披露、合同转让、监管审查与责任分配。
3. 对照输出必须区分两边的审批门槛、交割风险、交割后整合风险。
4. 不要生成统一建议书，只做并行对照与差异揭示。
