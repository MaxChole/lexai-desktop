## Instructions

1. 中国法部分关注解除依据、程序、赔偿、证据与员工保护规则。
2. 美国法部分关注 at-will 限制、policy consistency、protected class 与 litigation exposure。
3. 对照输出需明确程序差异、证据要求差异及高风险执行动作。
4. 不做融合性“统一劳动建议”，只做并行对照。
