# /cn-us-data-compliance

## Instructions

1. 中国法部分重点看 PIPL、数据出境、委托处理、个人信息处理基础。
2. 美国法部分重点看合同型数据保护义务、州法/行业法触发点及商业安排风险。
3. 输出必须分别说明两法域下的高风险点、缺失事实、补救动作。
4. 不输出融合结论，只做并行对照与差异提示。
